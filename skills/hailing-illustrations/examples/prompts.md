# 使用示例

## 只做配图规划

```text
Use $hailing-illustrations 先不要生图。
分析下面这篇中文文章，输出 5 张左右的 shot list。
每张写清放置位置、核心意思、结构类型、海灵动作、主隐喻、短标注和生成风险。

<粘贴文章>
```

## 直接生成正文配图

```text
Use $hailing-illustrations 把下面这篇文章生成 4 张正文配图。
要求：16:9、纯白背景、极简手绘、大量留白、海洋蓝与少量珊瑚橙，海灵必须承担每张图的核心动作。每张单独生成，不要拼图。

<粘贴文章>
```

## 生成 Odoo 一图看懂教程

```text
Use $hailing-illustrations 为“Odoo 从线索到收款的基础流程”生成一张轻教程总览图。
使用 4 个稀疏小场景：线索、报价、订单、收款。海灵在每一步执行动作，不要真实 UI，不要密集模块列表，保留大量白底。
```

## 为一个抽象判断生成单图

```text
Use $hailing-illustrations 为“稳定不是不变化，而是变化时仍有清晰航线”生成一张正文配图。
不要写标题。重新发明一个隐喻，让海灵承担核心动作，标注不超过 5 个短词。
```

## 修改现有图片

```text
Use $hailing-illustrations 编辑这张图：去掉错误标题，减少一半装饰浪花，保留海灵和正确流程，其他构图不变。
```
